# درمان EMDR - تجربه‌های ناگوار
# معالجة EMDR - التجارب المؤلمة

A professional web application for self-administered EMDR (Eye Movement Desensitization and Reprocessing) treatment with full support for **English**, **Farsi (فارسی)**, and **Arabic (العربية)**.

---

## 🌍 **زبان‌های پشتیبانی شده | اللغات المدعومة | Supported Languages**

- 🇬🇧 **English**
- 🇮🇷 **Farsi (فارسی)**
- 🇸🇦 **Arabic (العربية)**

---

## 📱 **ویژگی‌ها | المميزات | Features**

✅ **سه زبان کامل** | ثلاث لغات كاملة | Full tri-lingual support  
✅ **RTL Support** برای فارسی و عربی | دعم RTL للفارسية والعربية  
✅ **Responsive Design** - کار بر روی تمام دستگاه‌ها  
✅ **Offline Support** - استفاده بدون اینترنت  
✅ **سرعت قابل تنظیم** | السرعة القابلة للتعديل  
✅ **رنگ‌های قابل شخصی‌سازی** | الألوان المخصصة  
✅ **Keyboard Shortcuts**  

---

## 🚀 **نحوه استفاده | طريقة الاستخدام | How to Use**

### **1. نصب وابستگی‌ها | تثبيت التبعيات | Install Dependencies**
```bash
npm install
```

### **2. توسعه محلی | تطوير محلي | Development**
```bash
npm run watch
```

### **3. ساخت برای تولید | البناء للإنتاج | Production Build**
```bash
npm run prod
```

### **4. باز کردن در مرورگر | فتح في المتصفح | Open in Browser**
- Simply open `index.html` in your web browser
- یا استفاده از `npm run hot` برای hot-reload

---

## ⌨️ **میانبرهای صفحه‌کلید | اختصارات لوحة المفاتيح | Keyboard Shortcuts**

| Key | Action | اقدام |
|-----|--------|-------|
| **Enter** | Start/Stop | شروع/توقف |
| **C** | Toggle Circles | تغییر دایره‌ها |
| **S** | Toggle Sound | تغییر صدا |
